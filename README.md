# Weekly Budget

Minimal black & white weekly expense tracker PWA.

## Features

- Set weekly spending limit
- Add/delete expenses
- Quick add buttons
- Undo last deleted expense
- Automatic new week every Monday
- Local-only storage
- Installable on Android via Chrome once hosted on GitHub Pages

## GitHub Pages setup

1. Upload these files to the repository root:
   - `index.html`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon.svg`
   - `README.md`
2. Commit the files.
3. Go to repository `Settings` → `Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save.
7. Open `https://schmolchen.github.io/weekly-budget/` on your phone.
8. In Chrome, tap the three dots and choose `Install app` or `Add to Home screen`.
